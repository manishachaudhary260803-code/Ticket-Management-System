Problem:
We receive a hundreads of support email daily. our agent manaully read, classify and respond to each ticket.which is slow and leads to impresonal canned responses.

Solution:
Build a ticket management system that uses AI to automatically classify, repsond and route support ticket. delivering faster, more personalised responses to students while freeing up agent for complex issues.

Ticket Statuses:
- Open
- Resolved
- Cloased

Ticket Categories:
- General Qustions
- Technical Question
- Refund Request

User Roles
- **Admin**: Deployee with the system. Can create and manage agents
- **Agent**: Created by admin. Can view and manage tickets.

Features:
Receive support email and create tickets
auto generate human friendly responses using knowledbase
Ticket list with filter and sorting
Ticket details view
AI powered ticket classification
AI summaries
AI suggested reply
User Management  (admin only)
Dashboard to view and Manage all tickets
